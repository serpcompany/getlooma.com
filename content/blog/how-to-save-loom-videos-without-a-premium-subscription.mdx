## MP4 Format Enforcement Over WebM

When MP4 output is preferred, explicit format selection becomes necessary.

**Available Format Enumeration:**
```bash
yt-dlp -F "https://www.loom.com/share/c43a642f815f4378b6f80a889bb73d8d"
```

**Specific MP4 Selection:**
```bash
yt-dlp -f http-transcoded "https://www.loom.com/share/c43a642f815f4378b6f80a889bb73d8d"
```

**Outcome:** MP4 container selection supersedes default WebM preference.
